# minesweeper
A minesweeper in the terminal written in Golang with Bubbletea. Very WIP still


![Ã´fbylding](https://user-images.githubusercontent.com/15870306/168896014-43dfb214-56a4-4b1d-8673-ec04a1a4302a.png)


Controls are the following:

 * Move the cursor around with arrow keys.
 * Flag bombs with 'f'
 * Open/reveal cells with the `o`
 * Press q to quit.
